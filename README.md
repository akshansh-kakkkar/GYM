🏋️ Gym Fitness Website

A modern and responsive Gym / Fitness landing website built using pure HTML, CSS, and JavaScript.
This project focuses on creating a clean fitness-themed user interface with smooth layout, responsive design, and engaging visual sections.

🌐 Live Website: https://gym-ten-sepia.vercel.app/

✨ Features

🏃 Modern gym-themed UI design

📱 Fully responsive for mobile, tablet, and desktop

🎯 Smooth scrolling navigation

🎨 Clean typography and spacing

⚡ Fast loading static website

💪 Sections for workouts, fitness motivation, and call-to-action

🛠️ Built With

HTML5 — Structure

CSS3 — Styling & layout

JavaScript (Vanilla JS) — Interactivity

Vercel — Deployment
